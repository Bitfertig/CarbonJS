⌛ datebob.js

DateBobJS is an javascript which recreates the good parts of php class Carbon to have a nice and easy way to handle dates and times. DateBobJS combines two awesome javascripts (strtotime like datetime formats, date) into one single javascript.


Download

Installation

npm i @dipser/datebob.js
<script type="module">
import { DateBob } from "@dipser/datebob.js";
var datebob = DateBob.func; // Usage as function datebob('now')
console.log( datebob('now'), new DateBob('now') );
</script>
